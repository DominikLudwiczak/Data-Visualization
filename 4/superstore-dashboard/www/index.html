Help section

Welcome to the Help Section of the Dashboard!


Dashboard Section:
  • The Dashboard section provides an overview of key metrics and visualizations.
  • Explore the various boxes and charts to gain insights into the data.
Plots Section:
  • In the Plots section, you can find detailed visualizations and charts.
  • Each plot represents different aspects of the data.
  • Hover over the plots to view additional details or interact with specific data points.
Interaction:
  • Drop-down menus allow to change displayed statistics
  • At the top you can select year for which data is displayed
  • At the top you can select year to which some key statistics are compared
  • By clicking on a certain state all data will be displayed for selected state only (you can undo this by selecting same state again)
